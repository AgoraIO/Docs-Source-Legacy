This page provides release notes for the Agora RTC C# SDK, which runs on Windows.

## v3.6.2 

v3.6.2 was released on May 7, 2022. This is the first release of the RTC C# SDK.

#### Features

Developed upon RTC Windows SDK v3.6.2, this release covers the major features of the Windows SDK. For details, see [Release Notes for Windows](./release_windows_video?platform=Windows#v362).

#### Reference

Refer to the following documentations to integrate the SDK into your project, and add real-time engagement functionalities to the app.

- [Get Started with Voice Calling](https://docs.agora.io/en/Voice/start_call_audio_windows_csharp?platform=Windows)
- [Get Started with Video Calling](https://docs.agora.io/en/Video/start_call_windows_csharp?platform=Windows)
- [Get Started with Interactive Live Streaming Premium](https://docs.agora.io/en/Interactive%20Broadcast/start_live_windows_csharp?platform=Windows)
- [Get Started with Interactive Live Streaming Standard](https://docs.agora.io/en/live-streaming/start_live_standard_windows_csharp?platform=Windows)
- [API Reference](./API%20Reference/csharp/API/rtc_api_overview.html)

Agora provides an open-source [Agora CSharp API Example](https://github.com/AgoraIO/Agora-C_Sharp-SDK/tree/release/3.6.2/APIExample) on GitHub. You can try it out or refer to the source code.
