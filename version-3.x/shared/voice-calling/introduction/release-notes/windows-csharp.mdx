## v3.6.2 

v3.6.2 was released on May 7, 2022. This is the first release of the RTC C# SDK.

#### Features

Developed upon RTC Windows SDK v3.6.2, this release covers the major features of the Windows SDK. For details, see <Link _blank to="../introduction/release-notes?platform=windows-cpp">Release Notes for Windows</Link>.

Agora provides an open-source [Agora CSharp API Example](https://github.com/AgoraIO/Agora-C_Sharp-SDK/tree/release/3.6.2/APIExample) on GitHub. You can try it out or refer to the source code.
