This page provides the release notes for the Agora Cocos2d-x SDK.

The Agora Cocos2d-x SDK wraps the Agora <Vg k="VSDK" /> SDKs for Android and iOS, with which you can quickly implement real-time communication functionality in Cocos2d-x projects developed in C++.


## Known issues and limitations

#### Compatibility with iOS 15

As of v3.5.1, the SDK supports iOS 15. To fix crashes that occur on iOS 15, upgrade the SDK to v3.5.1 or later.

## v3.5.1

v3.5.1 was released on November 26, 2021.

#### Compatibility changes

**1. Compatibility with iOS 15 (iOS only)**

As of v3.5.1, the SDK supports iOS 15. To fix crashes that occur on iOS 15, upgrade the SDK to v3.5.1 or later.

**2. SDK version adaptation**

This release adapts to the Agora Voice SDK v3.5.1 for Android and iOS. For the compatibility changes, new features, improvements, and fixed issues from v3.2.0 to v3.5.1, see [Release Notes (Android)](release-notes) and [Release Notes (iOS)](elease-notes).

You can call C++ APIs to implement the corresponding functionality.

## v3.2.0

v3.2.0 was released on December 11, 2020.

#### Functions and features

- Adapts to Agora <Vg k="VSDK" /> v3.2.0.
- Provides C++ APIs to implement voice calls and interactive live audio streaming.
- Supports cross-platform application development for Android and iOS.
